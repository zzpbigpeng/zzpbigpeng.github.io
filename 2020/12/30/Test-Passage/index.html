node.js : 用于运行Hexo环境

  • npm 默认下载的包管理器

git : 用于将配置文件发布到运程仓库GitHub中，方便别人浏览。

  2. 安装git部署的插件：hexo-deployer-git
  1. 修改blog下的_config.yml文件
  1. 部署到远端
