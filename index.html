Hexo搭建博客环境

下载需要的软件

node.js : 用于运行Hexo环境

  • node 基于 Chrome V8 引擎的 JavaScript运行环境
  • npm 默认下载的包管理器

git : 用于将配置文件发布到运程仓库GitHub中，方便别人浏览。

正式开始搭建Hexo

将博客部署到GitHub上

  1. 新建一个仓库【命名规则：用户名.github.io】
  2. 安装git部署的插件：hexo-deployer-git
  1. 修改blog下的_config.yml文件
  1. 部署到远端
